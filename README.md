### Installation
1) Install the package *devtools*
2) Load *devtools* using `library(devtools)`.
3) Install *CPVS* using `install_github("whitenoise8/CPVS")`

### Reference
Bianco and Cappello (2026). "Computationally efficient segmentation for non-stationary time series with oscillatory patterns".
